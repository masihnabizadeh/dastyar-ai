# Dastyar Smart Shopping (Persian WooCommerce AI Assistant)

**Version:** 2.0.4  
**Author:** Honix Digital Solution  
**Language:** Persian (Farsi)

An intelligent Persian shopping assistant plugin for WooCommerce, designed to help users choose products dynamically through an AI chat interface.  
Supports **OpenAI (ChatGPT)** and **AvalAI.ir** providers.
