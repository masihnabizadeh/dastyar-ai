An AI-powered Persian shopping assistant for WooCommerce — now supporting both OpenAI and AvalAI providers.

== Description ==
Dastyar Smart Shopping is an AI-driven assistant plugin that helps Persian WooCommerce users interact with products through an intelligent chat interface.  
The plugin supports both **OpenAI** and **AvalAI** APIs for chat completions, offers onboarding, personalization, and dynamic greeting features.

== Features ==
* Chat-based shopping assistant
* Supports OpenAI & AvalAI
* Customizable persona and greeting templates
* Secure sanitization and rate limiting
* RTL Persian interface

== Installation ==
1. Upload the plugin to your WordPress installation.
2. Activate it from Plugins menu.
3. Go to “دستیار خرید” settings and add your API key.
